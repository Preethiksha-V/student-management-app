# Student Management System

Console-based Python application developed for  
ISWE406P – Agile Development Process and DevOps Lab  
(Winter Semester 2025–26).

## Features
- Add student details
- View all students
- Search student by roll number
- Grade calculation (added via feature branch)

## Tools Used
- Python
- Git
- GitHub

## DevOps Practices Followed
- Local and remote Git repositories
- Version control with commits
- Feature branching and merging
